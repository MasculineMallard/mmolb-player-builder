Layout Options (Round 2)

Each mockup is a 16:9 laptop viewport with Princess Solano Lv.21 batter, Clutch Performer archetype. Focus: information density, data visualization, zero scroll.

1 Heatmap Grid + Action Strip
Replace the stat table with a color-coded heatmap grid. Each stat is a colored cell (yellow=low, blue=high). You read the player at a glance. Top strip: identity + archetype + next action all in one line. Center: stat heatmap (left) + stat dev bars (right). Progression is a thin bar pinned to the bottom.
